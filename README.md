## Selfhosting
I'm not sure why you would selfhost this, but you can if you want. Make sure you have Node.js already installed.
```bash
$ git clone https://github.com/FireStreaker2/GuessTheError.git
$ cd GuessTheError
$ npm i
```
For developing:
```bash
$ npm run dev
```
For production:
```bash
$ npm run build
$ npm start
```
